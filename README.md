A comprehensive, production-ready GraphQL tutorial for junior developers. This repository contains complete working examples for building a full-stack GraphQL application with **React** (frontend) and **Go** (backend) running locally on macOS.

## ğŸš€ Quick Start

### Prerequisites

- **macOS** with zsh
- **Go** 1.21+
- **Node.js** 18+
- **Git**

### 1. Clone Repository

```bash
git clone https://github.com/nfsarch33/graphql-react-go-fullstack.git
cd graphql-react-go-fullstack
```

### 2. Backend Setup

```bash
cd backend

# Initialize Go module (if needed)
go mod download

# Run the server (starts on http://localhost:8080)
go run cmd/server/main.go
```

### 3. Frontend Setup (in new terminal)

```bash
cd frontend

# Install dependencies
npm install

# Start development server (runs on http://localhost:5173)
npm run dev
```

### 4. Access the Application

- **GraphQL Playground**: http://localhost:8080/graphql
- **React App**: http://localhost:5173

## ğŸ› Troubleshooting

| Issue | Solution |
|-------|----------|
| Port 8080 already in use | Change port in `backend/cmd/server/main.go` |
| Port 5173 already in use | Vite will auto-increment, or use `npm run dev -- --port 3000` |
| Module not found errors | Run `go mod tidy` in backend directory |
| Dependencies not installing | Clear node_modules: `rm -rf node_modules && npm install` |
| GraphQL schema not updating | Run `go run github.com/99designs/gqlgen generate` |
